# /deploy - Multi-User Deployment

Set up a new deployment or manage an existing one.

## Instructions

First, check if the user has an existing deployment. Ask:

> "Do you have an existing Hermes deployment, or is this a new setup?
> 1. **New deployment** — set up from scratch
> 2. **Manage existing** — add/remove users, check status, restart"

**If new deployment:** Read `deploy/onboarding.md` and follow the step-by-step onboarding instructions.

**If managing existing:** Ask for the deployment path, then read `deploy/managing.md` and follow the management instructions. Use the provided path as DEPLOY_PATH throughout.
