# /deploy - Multi-User Deployment Setup

Guide the user through setting up multi-user Hermes deployment with Docker containers.

## Instructions

Read `deploy/onboarding.md` and follow the step-by-step instructions there to guide the user through the full deployment setup process.

The onboarding doc covers:
1. Explaining what multi-user deployment does
2. Running setup.sh to scaffold the deployment directory
3. Configuring users, Mimir, and scopes conversationally
4. Writing the final config.yml
5. Running setup.sh again to generate everything
6. Verification and next steps
